# 用語集（Glossary）

| 用語 | 説明 |
| --- | --- |
| **Collector Layer** | Web/API/RSS など多様なソースからデータを取得する層。 |
| **Normalizer Layer** | 収集データを共通 JSON スキーマへ変換する層。 |
| **Storage & Analysis Layer** | Supabase / Qdrant / Notion を跨いで蓄積・活用する層。 |
| **Local SRS** | チーム単位で派生させる仕様書。Global SRS との差分を明示する必要がある。 |
| **Global SRS** | 本リポジトリで管理される正本（Single Source of Truth）。 |
| **Assurance Artefacts** | テスト戦略・CI/CD・検証結果など品質保証に関するドキュメント。 |
| **Governance Policy** | ブランチ運用、データ利用、アクセス権限を定義するルール群。 |

> Glossary は `docs/overview/glossary.md` を唯一の参照とし、略語を追加する際は pull request で更新してください。
